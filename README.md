#Tento projekt je fanúšikovský online obchod s produktmi Apple. Stránka obsahuje katalóg produktov (iPhone, MacBook, AirPods) s krátkym popisom. Používateľ si môže produkty prezerať a prejsť na nákup. Stránka nie je oficiálny obchod. Tlačidlá „Kúpiť“ presmerujú na stránky oficiálnych predajcov.



#Apple Store - PHP & MySQL (OOP + CRUD)

Tento projekt bol vytvorený ako semestrálna práca pre predmet Skriptovacie jazyky (PHP).
Projekt demonštruje prácu s databázou cez PDO a využíva plnohodnotné objektovo-orientované programovanie (OOP).
